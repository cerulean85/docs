# Windows 95: GUI 운영체제를 통일한 황제

<img src='/images/computer/windows95.png' width='20%' />

지금은 우리에게 너무나 익숙한 마이크로소프트의 Windows 운영체제는 사실 처음에는 애플 매킨토시의 운영체제의 GUI를 그대로 흉내내어 만들어졌습니다. GUI의 가치를 알아본 빌 게이츠는 애플에 GUI 일부분에 대한 라이선스를 허가 받아 1985년 Windows 1.0을 출시하였고, 1987년에는 매킨토시와 비슷한 GUI를 포함하여 Windows 2.0을 출시하였습니다.

<img src='/images/computer/windows1.jpeg' width='20%' />

<img src='/images/computer/windows2.jpeg' width='20%' />

애플은 매킨토시 운영체제의 룩앤필을 그대로 복제하였다는 이유로 마이크로소프트를 지적재산권 침해로 고소하였습니다. 하지만 법원에서는 189개 요소 중 179개는 마이크로소프트가 애플과의 합의사항과 벗어나지 않았으며, 나머지 10개는 지적재산권 보호범위를 침해하지 않았다고 판단하여 마이크로소프트에게 손을 들어줬습니다.

<img src='/images/computer/windows3.jpeg' width='20%' />


Windows는 3.1까지 MS-DOS 위에서 동작하는 응용 소프트웨어에 불과했으므로, MS-DOS가 가진 근본적인 문제 때문에 더이상 발전할 수가 없었습니다. 결국 빌 게이츠는 1992년 Windows 95 개발 계획을 수립하여 Windows 중심 운영체제 개발에 집중하기 시작하였으며, 1995년, Windows 95는 성공적으로 출시되어 세계적으로 큰 히트를 치게 됩니다.

<img src='/images/computer/windows95-2.webp' width='20%' />

<img src='/images/computer/windows95-3.jpeg' width='20%' />

Windows 95 사용화면을 보면 지금 우리가 사용하는 Windows의 모습과 거의 동일합니다. 그러니까 Windows 운영체제 사용자 인터페이스의 전반적인 모습은 이때 만들어졌다고 보면 됩니다. Windows 95는 선점형 멀티태스킹을 도입함으로써 다른 운영체제와의 차별점을 확보할 수 있었습니다.

우리나라에서는 때마침 PC 보급과 교육에 대한 필요성이 대두되고 있었던 터라 사회적 분위기에 맞춰 Windows 95가 설치된 PC가 전국적으로 보급되기 시작했습니다. 또한 Windows 95의 막강한 사용성은 독자적인 PC 생태계를 지니고 있었던 일본 마저 무너뜨리며 대히트를 기록했으며, 엑셀과 같은 마이로소프트 Office 제품 덕분에 사무실에도 Windows 95가 탑재된 PC가 급속도로 보급되기 시작했습니다. 이처럼 마이크로소프트는 Windows 95를 출시함으로써 이후 GUI 운영체제 시장에서 승자로 군림하게 됩니다. 덕분에 지금까지도 데스크톱 시장에서는 마이크로소프트의 Windows 운영체제가 독보적입니다.

Windows 95의 대히트는 IT시장에 큰 반향을 일으켰습니다. 무엇보다 당시 시장에는 운영체제가 다양했는데, 이 때문에 하드웨어 역시 운영체제에 맞게 여러 가지 방식으로 구성되어야 했습니다. 표준화가 되지 않아 호환성에 문제가 많았지만 Windows 95가 시장을 주도하며 하드웨어가 Windows 95를 중심으로 구성되며 표준화 되기 시작했습니다. 이는 이후 소프트웨어 발전에 큰 영향을 미친 대전환점이라고까지 평가되고 있으며, 우수한 개발도구와 훌륭한 소프트웨어를 만드는 기반을 제공하였습니다.
